# Git Lab

This project was created as part of a Git lab assignment.

## Tasks Completed

- Created a local Git repository.

## Git Commands Used

```bash
git init
git add Note
git commit -m "Add Note file"
git branch -M main
git remote add origin https://github.com/Mohamed-Hegazy0/GitLab.git
git push -u origin main
```
